- Initial release of BIR PH JMIT app
- Sales Register (SAR) report
- Purchase Register (PAR) report
- Monthly Tax Compliance report
- Summary of Sales, Purchases and Creditable Withholding Taxes (Form 2550M)
- Creditable Withholding Tax report
- Final Withholding Tax report
- Expanded Withholding Tax report (Form 2307)
- VAT Reconciliation report

- Support for ERPNext v15

- CAS Application compatibility
- Multiple export formats (CSV, PDF, Excel)
